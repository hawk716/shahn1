# دليل ميزة السحب

## الملفات المضافة:

### 1. ملف بيانات التطبيقات
**المسار:** `data/withdrawal-apps.json`
يحتوي على بيانات التطبيقات والمحافظ المتاحة للسحب.

### 2. مسار API
**المسار:** `app/api/withdrawal/request/route.ts`
يتعامل مع طلبات السحب ويتحقق من البيانات والرصيد.

### 3. مكون النموذج
**المسار:** `components/withdrawal/withdrawal-form.tsx`
واجهة المستخدم كاملة مع Modals.

## كيفية الاستخدام:

### في الصفحة الرئيسية:
```tsx
import { WithdrawalForm } from '@/components/withdrawal/withdrawal-form'

// داخل الصفحة بعد التحقق من دخول المستخدم
<WithdrawalForm userId={user.id} />
```

## الميزات:

✓ اختيار التطبيق مع الأيقونات
✓ اختيار العملة حسب التطبيق
✓ إدخال المبلغ مع التحقق من الحد الأدنى والأقصى
✓ إدخال رقم الحساب
✓ Modal تأكيد الطلب
✓ Modal إدخال كلمة المرور
✓ Modal رسالة النجاح
✓ إرسال بريد تأكيد
✓ خصم المبلغ من الرصيد

## التعديلات المطلوبة:

1. إضافة صور التطبيقات في `public/images/apps/`
2. تحديث بيانات التطبيقات في `data/withdrawal-apps.json`
3. إضافة المكون إلى الصفحة الرئيسية
